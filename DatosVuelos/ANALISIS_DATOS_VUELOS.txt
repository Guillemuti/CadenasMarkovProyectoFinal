APLICACIONES POTENCIALES PARA CADENAS DE MARKOV
==============================================================================

1. ANÁLISIS DE PUNTUALIDAD COMO PROCESO ESTOCÁSTICO:
   - Estados: {Puntual, Retrasado}
   - Analizar probabilidad de retraso dado historial de la ruta/aerolínea
   - Estimar matriz de transición P para diferentes rutas

2. ANÁLISIS DE RUTAS (ORIGEN-DESTINO):
   - Construir cadena de Markov de tráfico aéreo
   - Estados: Aeropuertos (identificados por ORIGIN/DEST_AIRPORT_ID)
   - Transiciones: Flujo de pasajeros entre aeropuertos
   - Calcular distribución estacionaria (aeropuertos más transitados)

3. ESTIMACIÓN DE COEFICIENTES β DE MEZCLA:
   - Aplicar estimador de McDonald (2015) a series temporales de retrasos
   - Variable: Indicador de retraso diario por aeropuerto/ruta
   - Verificar propiedad de β-mixing: dependencia temporal en retrasos
   - Comparar coeficientes β entre diferentes aeropuertos

4. SERIES TEMPORALES DE PUNTUALIDAD:
   - Analizar serie temporal diaria: X_t = proporción de retrasos en día t
   - Estimar β(a) para medir dependencia entre días separados por lag a
   - Comparar con estimador de S&P 500 del notebook actual

5. ANÁLISIS POR AEROPUERTO:
   - Identificar aeropuertos con mayor/menor dependencia temporal
   - Comparar coeficientes β entre aeropuertos hub vs regionales

6. MODELO GARCH PARA VARIABILIDAD DE RETRASOS:
   - Modelar volatilidad en retrasos (similar al análisis de S&P 500)
   - Usar GARCH para predecir períodos de alta variabilidad
   - Validar con estimador β de residuos

POSIBLES ANÁLISIS ESPECÍFICOS
==============================================================================

A. CADENA DE MARKOV DE ESTADOS DE PUNTUALIDAD:
   - Para cada ruta/aeropuerto, crear serie temporal binaria:
     X_t = 1 si vuelo en día t tiene retraso ≥15 min, 0 si no
   - Estimar matriz de transición 2×2:
     P = [P(Puntual|Puntual)    P(Retrasado|Puntual)  ]
         [P(Puntual|Retrasado)  P(Retrasado|Retrasado)]
   - Calcular distribución estacionaria π

B. RED DE TRÁFICO AÉREO:
   - Estados = Aeropuertos
   - P(i,j) = Proporción de vuelos desde aeropuerto i hacia j
   - Encontrar aeropuertos más centrales (PageRank)
   - Distribución estacionaria = tráfico esperado a largo plazo

C. ESTIMADOR β PARA DEPENDENCIA TEMPORAL:
   - Crear serie temporal: R_t = tasa de retraso en día t
   - Aplicar estimador McDonald (2015) con d=1, a=1,2,...,30
   - Graficar β^1(a) vs lag a
   - Verificar si β(a) → 0 (propiedad de mixing)

D. COMPARACIÓN ENTRE MESES:
   - Calcular coeficientes β para cada mes (Ene-Jul)
   - Identificar meses con mayor dependencia temporal
   - Relacionar con estacionalidad/eventos especiales
